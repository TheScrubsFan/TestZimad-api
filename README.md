# TestZimad API

## Getting Started

Run bundle (for fish syntax).


```
bundle install --path ~/.bundle --binstubs & git checkout bin/ & spring stop & spring binstub 
```

## Running

Start server. Available at http://localhost:4567/takeoff_time


```
ruby server.rb 
```
